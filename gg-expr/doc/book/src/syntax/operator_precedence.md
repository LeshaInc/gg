# Operator Precedence

| Operators                       | Associativity |
|---------------------------------|---------------|
| <code>\|\|</code>, `??`         | left to right |
| `&&`                            | left to right |
| `==`, `!=`                      | left to right |
| `<`, `<=`, `>=`, `>`            | left to right |
| `+`, `-`                        | left to right |
| `*`, `/`, `%`                   | left to right |
| unary `-`, `!`                  |               |
| `**`                            | right to left |
| call, index                     |               |
| `if`, `when`                    |               |
